# 🧾 项目概述

# 🎯 项目目标
实施项目所要达到的期望结果，即项目所能交付的成果或服务。 项目的实施过程实际就是一种追求预定目标的过程，因此，从一定意义上讲，项目目标应该是被清楚定义，并且可以是最终实现的。

1. 目标符合客户、用户的要求；
2. 目标回答了这样三个问题：做什么、在什么时候完成、用多少钱。

项目目标遵循 SMART 原则：
[向右] Specific 明确的
[向右] Measurable 可度量的
[向右] Achievable 可达到的
[向右] Result driven 结果驱动的
[向右] Timing 时间性
